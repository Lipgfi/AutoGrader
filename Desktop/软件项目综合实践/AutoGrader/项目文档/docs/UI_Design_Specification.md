# AutoGrader UI 设计规范 V1.1

## 一、设计原则

### 1.1 核心原则
- **一致性**：保持界面元素、交互方式、视觉风格的一致性
- **可用性**：确保用户能够快速理解和使用系统功能
- **可访问性**：支持不同设备和用户群体的访问需求
- **美观性**：符合学校VI规范，体现专业形象

### 1.2 设计理念
- 简洁明了，减少用户认知负担
- 高效操作，优化用户工作流程
- 友好反馈，提供清晰的操作指引

## 二、颜色体系

| 颜色名称 | 色值 | 使用场景 |
|---------|------|---------|
| 主色 Primary | #003D79 | 主要按钮、导航栏、重要标题 |
| 主色浅 Primary Light | #406A9C | 悬停状态、次要按钮 |
| 主色深 Primary Dark | #002A59 | 激活状态、强调元素 |

| 颜色名称 | 色值 | 使用场景 |

## 八、组件规范

### 8.1 按钮

#### 按钮类型
- **主要按钮**：主色调背景，用于主要操作
- **次要按钮**：白色背景+边框，用于次要操作
- **文字按钮**：无背景，用于辅助操作
- **危险按钮**：红色背景，用于删除等危险操作

#### 按钮尺寸

| 尺寸 | 高度 | 内边距 | 字体大小 |
|-----|------|-------|---------|
| Small | 24px | 8px | 12px |
| Default | 32px | 12px | 14px |
| Large | 40px | 16px | 16px |

#### 按钮状态
- **默认**：正常显示
- **悬停**：颜色加深 10%
- **激活**：颜色加深 20%
- **禁用**：透明度 50%

### 8.2 表单

#### 输入框
- 高度：32px (默认) / 40px (大)
- 边框：1px solid #DCDFE6
- 圆角：4px
- 聚焦边框：主色调

#### 选择器
- 与输入框保持一致的样式
- 下拉面板使用轻阴影

#### 表单标签
- 字体大小：14px
- 颜色：#606266
- 必填标记：红色星号

### 8.3 表格

#### 表头
- 背景：#F5F7FA
- 字体：14px，粗体
- 颜色：#303133

#### 表格行
- 高度：48px
- 边框：1px solid #EBEEF5
- 悬停背景：#F5F7FA

#### 分页
- 位置：右下角
- 显示：总条数 + 页码 + 跳转

### 8.4 卡片

#### 基础卡片
- 背景：白色
- 圆角：8px
- 阴影：轻阴影
- 内边距：16px

#### 卡片头部
- 高度：48px
- 边框：1px solid #EBEEF5
- 字体：16px，粗体

### 8.5 弹窗

#### 模态框
- 宽度：400px / 600px / 800px
- 圆角：12px
- 遮罩：rgba(0,0,0,0.5)
- 动画：淡入淡出 0.3s

#### 消息提示
- 位置：顶部居中
- 宽度：auto
- 动画：从上滑入

## 九、响应式设计

### 9.1 断点设置

| 设备类型 | 断点 | 说明 |
|---------|------|------|
| 手机 | < 768px | 小屏幕设备 |
| 平板 | 768px - 1024px | 中等屏幕设备 |
| 桌面 | > 1024px | 大屏幕设备 |

### 9.2 响应式规则

```css
/* 移动端优先 */
.container {
  padding: 16px;
}

/* 平板 */
@media (min-width: 768px) {
  .container {
    padding: 24px;
  }
}

/* 桌面 */
@media (min-width: 1024px) {
  .container {
    padding: 32px;
    max-width: 1200px;
  }
}
```

### 9.3 布局适配

#### 移动端 (< 768px)
- 单列布局
- 导航栏折叠
- 表格横向滚动
- 按钮全宽显示

#### 平板 (768px - 1024px)
- 双列布局
- 侧边栏折叠
- 表格自适应

#### 桌面 (> 1024px)
- 多列布局
- 完整导航
- 表格完整显示

## 十、交互动效

### 10.1 过渡动画

| 类型 | 时长 | 缓动函数 |
|-----|------|---------|
| 快速 | 0.2s | ease |
| 标准 | 0.3s | ease |
| 慢速 | 0.5s | ease |

### 10.2 常用动画

```css
/* 淡入淡出 */
.fade-enter-active,
.fade-leave-active {
  transition: opacity 0.3s ease;
}

/* 滑入滑出 */
.slide-enter-active,
.slide-leave-active {
  transition: transform 0.3s ease;
}

/* 缩放 */
.scale-enter-active,
.scale-leave-active {
  transition: transform 0.3s ease, opacity 0.3s ease;
}
```

### 10.3 加载状态
- 按钮加载：显示旋转图标
- 页面加载：骨架屏
- 数据加载：进度条

## 十一、无障碍设计

### 11.1 对比度
- 文字与背景对比度 ≥ 4.5:1
- 大文字对比度 ≥ 3:1

### 11.2 焦点状态
- 明显的焦点边框
- 键盘导航支持

### 11.3 语义化标签
- 使用正确的 HTML 标签
- 添加 ARIA 属性

## 十二、设计交付

### 12.1 设计稿命名规范
- 格式：模块_页面_状态
- 示例：Login_Index_Default

### 12.2 切图规范
- 格式：PNG / SVG
- 倍率：@1x, @2x, @3x
- 命名：icon_名称_尺寸

### 12.3 标注规范
- 使用 px 单位
- 标注相对位置
- 说明交互效果
