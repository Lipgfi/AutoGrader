# Excel模板格式和权限规则配置

## 一、学生信息批量导入Excel模板格式

### 模板结构

| 列名 | 数据类型 | 说明 | 是否必填 | 验证规则 |
|------|---------|------|----------|----------|
| 学号 | 文本 | 学生唯一标识 | 是 | 长度10-20位，字母数字组合 |
| 姓名 | 文本 | 学生姓名 | 是 | 长度2-20个字符 |
| 用户名 | 文本 | 登录用户名 | 是 | 长度4-20位，字母数字下划线 |
| 邮箱 | 文本 | 学生邮箱 | 是 | 有效的邮箱格式 |
| 手机号 | 文本 | 学生手机号 | 是 | 11位数字 |
| 专业 | 文本 | 学生专业 | 是 | 下拉选择：计算机科学与技术、软件工程、数据科学与大数据技术 |
| 年级 | 文本 | 学生年级 | 是 | 下拉选择：2023级、2024级、2025级 |
| 状态 | 文本 | 学生状态 | 否 | 激活/禁用，默认激活 |

### 模板示例

```
学号,姓名,用户名,邮箱,手机号,专业,年级,状态
S001,张三,student1,zhangsan@example.com,13800138001,计算机科学与技术,2023级,激活
S002,李四,student2,lisi@example.com,13800138002,软件工程,2023级,激活
S003,王五,student3,wangwu@example.com,13800138003,数据科学与大数据技术,2023级,激活
```

### 导入验证规则

1. **必填字段验证**：学号、姓名、用户名、邮箱、手机号、专业、年级为必填
2. **格式验证**：
   - 学号：长度10-20位，字母数字组合
   - 邮箱：符合邮箱格式
   - 手机号：11位数字
3. **唯一性验证**：学号和用户名必须唯一
4. **下拉选项验证**：专业和年级必须在指定范围内

## 二、权限规则

### 权限级别

| 级别 | 代码 | 说明 |
|------|------|------|
| 个人 | personal | 只能查看和编辑自己的信息 |
| 班级 | class | 可以管理本班级所有学生信息 |
| 年级 | grade | 可以管理本年级所有学生信息 |
| 管理员 | admin | 可以管理所有学生信息 |

### 操作权限矩阵

| 操作 | personal | class | grade | admin |
|------|----------|-------|-------|-------|
| 查看学生列表 | ✓ | ✓ | ✓ | ✓ |
| 查看学生详情 | ✓ | ✓ | ✓ | ✓ |
| 添加学生 | ✗ | ✗ | ✓ | ✓ |
| 编辑学生 | ✓ | ✓ | ✓ | ✓ |
| 删除学生 | ✗ | ✗ | ✓ | ✓ |
| 批量导入 | ✗ | ✗ | ✓ | ✓ |
| 批量导出 | ✗ | ✓ | ✓ | ✓ |
| 状态管理 | ✓ | ✓ | ✓ | ✓ |

### 权限控制实现

1. **前端权限控制**：
   - 基于用户角色显示/隐藏操作按钮
   - 基于权限级别控制操作可用性
   - 实现权限检查函数，用于判断用户是否有操作权限

2. **后端权限控制**：
   - API接口权限验证
   - 数据访问权限控制
   - 操作日志记录

## 三、使用说明

### Excel模板使用

1. 下载模板文件
2. 按照模板格式填写学生信息
3. 保存为.xlsx或.xls格式
4. 在系统中上传文件进行导入
5. 查看导入结果，处理失败记录

### 权限管理

1. 管理员在系统设置中配置用户权限
2. 用户登录后，系统根据权限级别显示相应功能
3. 操作时系统自动检查权限，无权限操作会被拒绝

## 四、注意事项

1. Excel文件大小建议不超过10MB，避免导入性能问题
2. 单次导入建议不超过1000条记录
3. 确保Excel文件编码为UTF-8
4. 权限设置应遵循最小权限原则
5. 定期备份权限配置
